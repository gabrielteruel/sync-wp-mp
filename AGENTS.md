# Contexto del Proyecto
Este es un proyecto Laravel 12.
- Framework: Laravel 12
- PHP Version: 8.3
- Database: SQLite 
- Frontend: sin frontend

# Instrucciones de Comportamiento
1. **Configuración**: Siempre asume que el entorno es fresco. Usa SQLite para la base de datos.
2. **Tests**: Antes de finalizar cualquier tarea, corre `php artisan test` para asegurar que no rompiste nada.
3. **Estilo**: Sigue el estándar PSR-12 para PHP.
4. **Rutas**: Las rutas web están en `routes/web.php`. No crees controladores invocables a menos que sea estrictamente necesario.

# Comandos Frecuentes
- Servidor local: `php artisan serve`
- Migraciones: `php artisan migrate`
- Tests: `php artisan test`
